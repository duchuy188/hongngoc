# Hồng Ngọc Textile Website

A static HTML/CSS/JavaScript website for Hồng Ngọc Textile company.

## Structure

```
hongngoc/
├── index.html          # Main HTML file
├── styles.css          # Stylesheet
├── script.js           # JavaScript functionality
├── assets/
│   └── images/         # Image assets folder
└── README.md           # This file
```

## Features

- Responsive design
- Image slider with auto-play
- Dropdown navigation menu
- Smooth scrolling
- Mobile-friendly layout
- Product showcase sections
- Language switcher (VI/EN)

## Setup

1. Add your images to the `assets/images/` folder with the following names:

   - slider1.jpg, slider2.jpg, slider3.jpg, slider4.jpg, slider5.jpg, slider6.jpg
   - product-soi.jpg
   - product-vai-moc.jpg
   - product-vai-tp.jpg
   - product-may-mac.jpg

2. Open `index.html` in your web browser

## Usage

Simply open the `index.html` file in any modern web browser to view the website.

## Browser Support
